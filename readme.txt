1. 터미널을 켭니다. -> command + space (검색창) terminal 검색 실행
2. 해당 폴더가 있는곳으로 위치를 이동하는 명령어를 칩니다. $ cd /Users/spryfit../../InstaAutoFollow
3. cd는 change directory 라는 뜻입니다.
4. 경로를 찾기 어려울땐 해당폴터를 드래그해서 터미널위에 올리면 경로가 나옵니다.
5. 터미널에 node app.js 를 입력합니다
6. 구글크롬을 실행해 주소창에 localhost:3001 로 접속하여 사용합니다.
7. 종료를 하고싶을땐 터미널에서 ctr + c 를 누릅니다.
8. 다시 시작하고 싶을땐 경로이동 후 node app.js를 입력합니다.
